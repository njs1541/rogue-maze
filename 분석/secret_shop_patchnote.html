System Analysis & Specs

네온 암시장 비밀 상점 시스템 분석

미로에 숨겨진 균열 벽 뒤에서 조용히 맥동하는 차원의 거래 기류. 최대 체력(Max HP)을 제물로 바치는 피의 서약 시스템 설계 사상 및 패치노트입니다.

🔮 1. 시스템 기획 사상 (Design Philosophy)

단순히 미로 속에서 보물 상자를 찾아 골드를 보상받는 일차원적 구조에서 탈피하여, 로그라이크 장르(예: 아이작의 번제, 엔터 더 건전)의 긴장감 높은 전통 문법인 "리스크 & 리턴"의 피의 계약 메커니즘을 Rogue-Maze에 결합했습니다.

플레이어는 코인 대신 자신의 생명력에 직결된 최대 체력(Max HP)의 영구 희생을 제물로 바쳐, 오직 암시장에서만 획득 가능한 최상위 등급(에픽/레전더리) 능력을 확정 거래하게 됩니다. 이는 플레이어가 현재 남은 체력 상황과 덱의 강력함을 끊임없이 재며 선택의 기로에 서게 만드는 깊이 있는 의사결정을 유도합니다.

비밀 균열 벽 타격 파괴
➔
50% 확률로 암시장 자판기 노출
↓

↓

[계약 거절] 무해한 모달 해제 및 자판기 보존

🛠️ 2. 주요 구현 상세 (Implementation Specs)

👾 2.1. 암시장 비밀 자판기 엔티티 (`SecretVendingMachine`)

  • 보랏빛 네온 시각 비주얼: 일반 자판기와 달리, 어두운 암시장 고유의 테마를 표현하기 위해 보라빛 네온 테마 컬러(`#b026ff`) 바디와 펄스 맥동 원형 후광을 드로잉합니다.
  • 차원 글로우 파티클: 15프레임 주기마다 기계 전역에서 보랏빛 글로우 파티클이 2개씩 스며 나오도록 처리하여 밀실 내의 신비롭고 어두운 존재감을 시각화했습니다.
  • 렌즈 스크린 및 가격 경고: 중앙 전광판에 미지의 카드를 상징하는 `🔮 ??` 문양과 하단에 피의 대가를 경고하는 `❤️ -15 HP` 경고 문구가 항시 출력됩니다.

⚡ 2.2. 무결성 스폰 제어 & 상자 겹침 버그 완벽 예외 처리

이전 빌드에서는 방 진입 단계에서 보물 상자(`HiddenChest`)가 무조건 스폰되어, 벽을 깼을 때 자판기와 상자가 한 좌표에 중첩되어 비주얼이 깨지고 이중 보상을 획득하는 결함이 발생할 위험이 있었습니다.

  • 밀실 대체 스폰 구조: 비밀 벽(`SecretWall`) 파괴 시 50% 확률을 추첨하여, 기존에 해당 밀실에 대기 중이던 보물 상자(`HiddenChest`)를 실시간으로 탐색해 배열에서 완벽히 제거(`splice`)합니다.
  • 상자 위치로 정밀 배치: 제거된 상자의 좌표(`spawnY`)를 그대로 계승해 그 밀실 자리에 `SecretVendingMachine`을 스폰함으로써 겹침 문제를 완벽 해결하고 기획 의도에 100% 부합하게 정렬했습니다.

🛡️ 2.3. 최대 체력 고갈 방지 안전 가드 (Anti-Death Guard)

플레이어가 거래 대가인 15보다 적거나 같은 최대 체력 상태(예: Max HP가 15 이하)에서 자판기를 작동해 스스로 사망해버리는 소프트락(Soft-lock)을 원천 방어합니다.

  • 제물 부족 검증: 충돌 순간 `player.maxHp <= 15` 조건 체크 시, 거래 모달을 띄우지 않고 쿨다운을 먹이며 ⚠️ 제물이 부족합니다 (Max HP ≤ 15) 경고 텍스트를 머리 위에 띄웁니다.
  • 넉백 튕겨내기: 플레이어에게 피격 틱 사운드와 함께 자판기 반대 방향으로 넉백 물리 반사를 가해 연속 마찰 충돌을 안전하게 방어합니다.

✨ 3. 차원 거래 처리 및 연출 (Transaction & VFX)

자판기 들이받기 성공 시 게임 시간이 완전히 동결되며 전용 모달인 [차원의 암시장 계약 오버레이]가 팝업됩니다.

🔮 서약 수락 (Accept Deal)

플레이어의 생명을 바쳐 차원의 무한한 지배 능력을 전수받습니다.

  • 플레이어 최대 체력 Max HP -15 영구 차감
  • 현재 체력이 최대치를 초과하지 않도록 즉시 조율 보정
  • 최상위 등급(Epic/Legendary) 1장 즉시 영구 장착
  • 차원 붕괴 파티클: 보랏빛 스파크 40개 및 다크 그레이 분진 파편 15개가 사방으로 폭발적 방출
  • 천둥 치는 폭발 사운드 재생 및 강한 화면 흔들림
  • 비밀 자판기는 소멸 제거 처리

⚙️ 계약 거절 (Reject Deal)

이계 상인의 달콤하고 위험한 꼬드김을 외면하고 안전을 챙깁니다.

  • 어떠한 최대 체력/현재 체력 삭감도 발생하지 않음
  • `DEAL REJECTED` 플로팅 중립 텍스트 생성
  • 경쾌한 회피 효과음 출력 및 시간 동결 즉시 해제
  • 자판기 상태 유지: 기계가 그대로 밀실에 남아있어, 방을 나가지 않았다면 언제든 다시 돌아와 계약 가능

💻 4. 주요 코드 아키텍처 (Code Highlights)

아래는 `game.js` 엔진에 구현된 비밀 상점 겹침 탐색 제거 알고리즘과 충돌/안전 가드 제어부 핵심 소스코드입니다.

⚙️ 4.1. 스폰 겹침 제거 및 밀실 대체 스폰

// [네온 암시장] 50% 확률로 보물 상자 대신 비밀 자판기 스폰
if (Math.random() < 0.5) {
    // 50px 이내에 대기 중이던 HiddenChest를 정밀 추적해 완벽 삭제
    for (let k = this.hiddenChests.length - 1; k >= 0; k--) {
        let chest = this.hiddenChests[k];
        if (Math.hypot(wall.x - chest.x, wall.y - chest.y) < 50) {
            this.hiddenChests.splice(k, 1);
            break;
        }
    }
    // 밀실 중심 좌표(spawnY) 계산 후 자판기 스폰
    let spawnY = wall.y < 300 ? wall.y - 40 : wall.y + 40;
    this.secretVendingMachines.push(new SecretVendingMachine(wall.x, spawnY));
    this.showFloatingText("🔮 SECRET SHOP APPEARED!", wall.x, spawnY - 20, '#b026ff');
}

⚙️ 4.2. 안전 가드 및 충돌 바인딩

// 비밀 자판기 충돌 감지 루프
let dist = Math.hypot(this.player.x - svm.x, this.player.y - svm.y);
if (dist < this.player.radius + 26) {
    if (this.vendingCooldown === 0) {
        // 최대 체력이 15 이하면 거래 불가 (안전 가드)
        if (this.player.maxHp <= 15) {
            this.vendingCooldown = 60;
            Sound.play('hit');
            this.showFloatingText("⚠️ 제물이 부족합니다 (Max HP ≤ 15)", this.player.x, this.player.y - 30, '#ff0055');
        } else {
            this.vendingCooldown = 60;
            // 반대 방향 튕겨냄 넉백 공식
            let bounceAngle = Math.atan2(this.player.y - svm.y, this.player.x - svm.x);
            this.player.x += Math.cos(bounceAngle) * 38;
            this.player.y += Math.sin(bounceAngle) * 38;
            
            // 차원 거래 수락/거절 핸들러 팝업 호출
            this.triggerSecretShopPurchase(svm, i);
        }
    }
}

🧪 5. 밸런스 팩터 및 검증 가이드 (QA Guide)

성공적인 빌드 릴리즈를 위해 직접 미로 내부에서 검증을 마칠 수 있는 시나리오 가이드라인입니다.

  • TEST 1 디버그 치트 시동: 게임 실행 후 `P`키를 눌러 치트 레이어를 활성화한 뒤, 비밀 균열 벽이 보일 때까지 방을 돌파합니다.
  • TEST 2 균열 벽 파괴: 마우스 클릭으로 검 베기를 가해 벽을 무너트렸을 때, 50% 확률로 상자가 사라지고 보랏빛 파티클 분수와 함께 비밀 자판기가 정확한 밀실 위치에 등장하는지 확인합니다.
  • TEST 3 체력 부족 테스트: 치트로 체력을 일부러 깎거나 15 이하의 상태를 만든 후 자판기를 들이받아 봅니다. 거래 창이 뜨지 않고 플레이어가 뒤로 안전히 넉백되며 머리 위에 `제물이 부족합니다` 경고가 생기면 통과입니다.
  • TEST 4 계약 분기 테스트: 체력이 16 이상인 상태로 들이받아 모달을 연 뒤 `계약 거절`과 `서약 수락`을 각각 눌러 봅니다. 거절 시 피해가 없고 수락 시 Max HP 가 즉각 15 깎이고 보라색 대규모 파티클 폭발 및 카드 디테일 팝업이 순차적으로 뜨며 자판기가 소멸하면 정상 연동이 완수된 것입니다.
