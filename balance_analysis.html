Comprehensive Game Design & Balancer Report

NEON ROGUE-MAZE 밸런스 종합 분석서

현재 개발된 game.js의 플레이어 빌드 카드, 성장 곡선, 몬스터 스케일링 데이터를 수학적 분석과 함께 시각화하고, 타 로그라이트 명작 게임들과의 비교분석을 통해 극단적 밸런스 이슈의 해결 패치를 제안합니다.

⚔️ 플레이어 빌드 시너지와 DPS 스노우볼링

Player Balance

현재 플레이어는 힘(ATK), 지능/공속(aspd), 사거리(RNG), 회피(EVD), 최대체력(HP), 재생(REG), 운(LUK) 등 다채로운 스탯과 함께, 5의 배수 방마다 특수한 상위 카드로 멀티샷(multishot), 점사(burstCount), 관통, 유도(homing), 스플래시(splash), 검(sword), 디펜더 펫(pet)을 획득할 수 있습니다.

분석 결과, 플레이어가 특정 카드 조합을 완성했을 때 발동하는 곱연산 빌드 시너지(Neon Bomber 1.8배, Neon Tempest 1.5배)와 무기 속성 변환(검기 투사체 관통 99개 동시 사출)이 기하급수적으로 중첩되면서, 후반부(80~100스테이지) 플레이어의 순간 폭발적인 DPS는 기계식 치트를 쓴 수준까지 스노우볼이 굴러갑니다.

⚠️ 플레이어 빌드의 결정적 밸런스 구멍 (3대 OP 요소)

  • 운(LUK) 스탯의 미친 스노우볼: LUK이 카드로 올라갈수록 portalLuckBonus = monsterBonus * 0.02 + (this.player.luk - 1) * 0.12 공식에 의해 EPIC/LEGENDARY 카드가 뽑힐 확률이 복리로 수렴합니다. LUK이 5 이상만 되어도 일반 등급 카드는 아예 안 나오며, 모든 보상이 최고 등급 카드로 도배됩니다.
  • 회피율(EVD) 75% 캡의 절대 생존력: EVD가 75%에 이르게 되면, 4번의 피격 중 3번을 완전 무시하며 DODGE 잔상을 남깁니다. 초당 자연 체력 재생(REG)과 결합될 시 무적이 됩니다.
  • 검기(Sword Blade) + 멀티샷 시너지의 광역 쓸어담기: 주무기 유형을 검(Sword)으로 융합하면 pierce: 99(무한 관통) 속성의 쾌검풍 투사체가 multishot 수만큼 부채꼴로 뿜어져 나갑니다. 부채꼴 퍼짐 각도 내의 모든 몬스터가 원샷에 녹아 전투가 극도로 단순화됩니다.

📈 플레이어 빌드 완성도에 따른 이론적 DPS 곡선

Interactive Simulation

* 일반 빌드: 시너지 융합 없이 힘/공속 카드만 선별한 선형적인 성장 라인.
* 시너지 빌드 (OP): 멀티샷 + 유도 + 스플래시가 맞물리며 Neon Bomber 시너지를 터뜨리고 검기 난무로 무쌍을 찍는 초스노우볼 성장 라인. 50스테이지를 기점으로 기하급수적으로 폭발합니다.

👾 몬스터 스케일링의 극단적 인플레이션

Monster Balance

현재 몬스터의 능력치(HP, ATK, SPD)는 방 번호에 따른 roomFactor = 1.0 + (roomNum - 1) * 0.05 단리 증가와 5번 방마다 오르는 몬스터 등급 티어(tier)에 의해 이중 복합 스케일링됩니다.

이중 연산 공식으로 인해, 후반부인 90~100스테이지에 들어서면 일반 몬스터들조차 공격력이 256, 최종 보스는 HP 7735, ATK 566이라는 초인플레이션 수치에 도달합니다.

💥 몬스터 밸런스의 치명적 설계 오류 (유리 대포 현상)

플레이어의 최대 HP는 최고급 보상 카드를 모아도 200~300을 넘기 어려운데, 후반부 일반 적의 평타 1대 공격력은 256입니다. 이는 "스치기만 해도 즉사"하는 원샷 원킬 지옥을 형성합니다.

플레이어가 만일 EVD 75%와 Homing/Splash Neon Bomber 시너지를 완성하지 못했다면(소위 '똥망 빌드'), 압도적인 체력을 지닌 보스(HP 7735)가 뿜는 부채꼴 즉사 탄막에 숨도 쉬지 못하고 비명횡사하게 됩니다. 반대로 사기 시너지를 맞추면 보스조차 5초 안에 녹이는 극단적 모 아니면 도 구조입니다.

🏆 유사 로그라이트 3대장 명작과의 밸런스 비교

Competitive Analysis

로그라이트 및 Roguelike Bullet-Hell 장르의 전설적인 타이틀들과 네온 로그메이즈의 설계 구조 및 밸런스 팩터를 5개 주요 항목으로 비교분석했습니다.

게임명 성장 체감 피지컬 난이도 스탯 인플레이션 빌드 시너지 깊이 밸런스 안정성
Vampire Survivors ★ 5.0 (압도적) ★ 1.5 (매우 낮음) 완만함 (30분 한계) 조합식 중심 대단히 안정적
Enter the Gungeon ★ 3.0 (도구 보강) ★ 4.8 (피지컬 지옥) 없음 (하트 한계제) 총기 복합 시너지 피지컬 축 기반 조화
Brotato ★ 4.5 (다양성) ★ 3.0 (무빙 강조) 중간 (웨이브 제한) ★ 4.5 (매우 다양) 하드캡 및 점감률 적용

핵심 시사점: 명작 게임들은 빌드의 강력한 쾌감을 보증하면서도 스탯 하드캡(Brotato 회피 60% 제한, Gungeon 고정 데미지 등)이나 몬스터 체력/데미지의 스펙 한계선을 뚜렷이 둡니다. 현재 네온 로그메이즈는 이러한 브레이크가 없어 후반부 억까 난이도를 연출합니다.

🎯 밸런스 5대 레이더 맵

Radar Radar

🛠️ 밸런스 복원 및 시스템 개선 솔루션 제안

Solution Code Patch

네온 로그메이즈의 원초적인 컨트롤 재미와 쾌감 넘치는 네온 탄막 액션을 100스테이지까지 조화롭게 살려내기 위해, 수치 점감 및 스케일링 상한선을 적용하는 3가지 코드 패치 솔루션을 제안합니다.

💡 핵심 패치 방향 3가지

  • 1. 몬스터 공격력 스케일링 공식 수정 (단리 상한선 적용): roomFactor가 5%씩 무한 상승하던 것을, 스테이지 후반부에는 점진적으로 스케일링이 누그러지도록 로그(Log)형 증가 또는 80% 한계 성장 캡을 씌웁니다.
  • 2. 플레이어 회피율 EVD 및 운 LUK 소프트 캡(점감 방식) 전환: 카드를 집을수록 무조건 더해지던 복리 덧셈 방식에서, 스탯이 올라갈수록 실질 효율이 줄어드는 체감 수렴 공식을 도입합니다. (회피 최대 55% 캡)
  • 3. 하이브리드 검기 사정거리 및 관통 횟수 캡 씌우기: 검기 투사체의 관통 횟수를 99회에서 burstCount 및 aspd에 비례하여 최대 3~6회로 차등 캡 제한을 두어 "검기 한 발로 방 쓸어담기" 억까를 방어합니다.

추천 개선 스케일링 코드 (game.js 반영 가이드)

// 1. 몬스터 능력치 이중 상승을 조율하는 안전 스케일링 공식 (Monster 클래스 적용)
let roomFactor = 1.0 + Math.log10(roomNum) * 1.5; // 무한 5% 단리에서 로그 스케일링으로 변경
this.atk = Math.ceil((5 + (tier - 1) * 1.5) * roomFactor); // 100스테이지 기준 공격력 256 -> 78 수준으로 안정화

// 2. 플레이어 회피율 EVD 점감 효과 공식 (applyRewardCard 메서드 적용)
case 'evd':
    // 복리 덧셈에서 '한계 효용 체감 공식' 적용 (최대 50% 수렴 캡)
    p.evd = p.evd + card.effectValue * (0.5 - p.evd); 
    break;

// 3. 검기(Sword Wave) 융합 투사체 관통 캡 제한 (slashWeapon 메서드 적용)
this.bullets.push(new Bullet(this.player.x, this.player.y, vx, vy, swordDmg, true, {
    pierce: Math.min(4, this.player.pierceCount + 1), // 무한 99회 관통에서 최대 4~5회 관통 캡 적용
    homing: this.player.homing,
    splash: this.player.splashRadius,
    color: '#b026ff',
    radius: 5
}));
