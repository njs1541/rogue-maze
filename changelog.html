미로를 탈출하기 위한 개발 과정과 릴리즈 마일스톤을 기록하는 공식 로그북입니다.

🚀 v0.3 구현 완료 (Stable)
2026-05-27 (완료)
🛠️ 계획된 구현 목표 (Changelog v0.3)
  • 모델 C 기반 전략 빌드 시너지 시스템 탑재: 4가지 특수 세트 버프 시너지(탄막 폭격기, 뇌신 검사 등) 대미지 곱연산 공식 추가 및 인게임 시각 피드백 연출.
  • 포털 랭킹 등급화 및 엘리트 드로잉: 포털 3곳의 난이도를 상/중/하(high, mid, low)로 실시간 등급화하고 5의 배수 진입 직전에 상/중 포털에 [ELITE] 캔버스 드로잉 추가.
  • 엘리트 몬스터 출현 기믹: 1.5배 거대 크기, 맹독 파편 비산 피드백, 체력 3배, 공격력 2배, 처치 보상 5배의 전용 초록 네온 엘리트 몬스터 구현.
  • 보상 시스템 전면 세분화: 일반 스테이지는 오직 캐릭터 기본 체질 카드(statusPool)만 스폰, 5/10의 배수 방은 상위 카드(weaponPool) 확정 스폰 및 탔던 문 난이도 비례 등급 보정(Epic/Legendary 확률 상승).
  • 스탯 밸런스 튜닝: 화력 급상승 억제를 위해 공격력(+3 ➔ +2), 공속(+0.2 ➔ +0.12) 등 기본 스탯 상승폭을 미세하게 하향 튜닝.

💎 v0.2 안정 빌드 (Stable)
2026-05-27 (이전 작업)
✨ 핵심 반영 사항 (Changelog v0.2)
  • 스폰 제어 3종 패턴 엔진 통합: 몬스터 10마리 누적 소환 시 스폰 속도가 지연되거나(방식 2), 전부 소탕할 때까지 다음 묶음 소환을 보류하는(방식 3) 등 전투 페이스 메이커 탑재.
  • Web Audio API 사음 합성 엔진: 별도의 오디오 파일 로드 없이 삼각파/사인파 노이즈 필터를 실시간 오실레이터로 합성하여 레트로 사운드(Laser Shoot, Slash, Dodge, Explosion, GameOver, Victory) 즉석 음향 지원.
  • 마우스 커서 동기화: 이등변삼각형 모양 본체의 마우스 조향 회전 궤적 정확화 및 60fps 매끄러운 캔버스 리프레시 구현.
