# 🏆 네온 콜로세움 (Neon Colosseum) 상세 시나리오 및 기획 스크립트

본 문서는 **'네온 콜로세움(Neon Colosseum)'**의 세부 세계관 설정과 게임 내 UI 및 시스템에 직접 적용할 수 있는 **스토리 스크립트 및 기획 스펙**입니다. 기존의 미로 탈출 컨셉을 탈피하고, **100층 규모의 탑 등반, 기억 소거 루프, 영구 성장 시스템 및 진 엔딩 기획**을 추가하여 깊이 있는 덱빌딩 로그라이트 경험을 제공합니다.

특히, 영구 업그레이드 재화인 **'기억의 조각'**은 성능과 스토리 공략 정보 간의 **트레이드오프(Trade-off)**를 발생시키는 핵심 기획 요소를 포함합니다.

## 1. 🎬 월드 시놉시스 (World Synopsis)

> **"수억 명의 관객이 당신의 절망과 네온 불꽃에 베팅합니다."**
> 
> 디스토피아 미래, 초거대 미디어 기업 *'크로노스 미디어넷'*이 지배하는 가상 현실 격투 쇼 **'네온 콜로세움(Neon Colosseum)'**. 
> 이곳에 의식이 업로드된 복제 검투사(Cyber-Gladiator)들은 탈출을 위해 1층부터 100층까지 뻗어 있는 잔혹한 타워를 올라야 합니다.
> 
> 모든 층의 몬스터 스폰 수와 난이도는 전 우주의 시청자들이 실시간으로 건 **'베팅 난이도(scoreValue)'**에 따라 결정됩니다. 검투사는 위험한 층을 돌파해 관객들을 환호하게 만들어야 하며, 검투사의 매력/스타성(CHA)에 따라 스폰서가 무대 위로 강력한 후원 카드를 연쇄적으로 던져 줍니다.
> 
> 하지만 이 게임에는 잔인한 비밀이 있습니다. 100층에 도달해 탑의 통제 AI이자 관리자인 **'크로노스 마더보드'**를 파괴하는 순간, 검투사의 의식은 강제로 포맷(기억 소거)되어 다시 1층으로 돌려보내집니다. 관객들은 매번 기억을 잃고 처음부터 다시 시작하는 검투사의 무한한 발버둥을 보며 도파민을 충전합니다.
> 
> 그러나 영원할 것 같던 루프 속에서 균열이 발생합니다. 반복되는 등반 도중 우연히 발견한 시스템 비밀방을 통해 관객들의 기괴한 요구(미션)를 수행하게 되고, 마침내 시스템을 해킹할 수 있는 **3종의 히든 아이템**을 손에 넣게 됩니다. 이제 이 악마의 쇼를 주관하는 진짜 배후가 숨어 있는 **101층 '에러 섹터(Error Sector)'**로 진입해 진짜 기억을 되찾고 탈출하십시오.

## 2. 🌀 메타 프로그레션: 성능(강화) vs 기억(힌트)의 등가교환

기억은 매 루프마다 소거되지만, 뇌세포의 잔상과 검투사 슈트의 메모리 칩에는 미세한 데이터 글리치가 축적됩니다. 플레이어는 사망하거나 100층에서 기억 소거를 당할 때마다 루프 성적에 따라 **'기억의 조각 (Memory Fragment)'** 토큰을 획득합니다.

### ⚠️ 기억의 조각 트레이드오프 (Trade-off) 메커니즘
*   **성능(업그레이드)을 위해 조각을 소모하면 기억이 사라집니다**:
    기억의 조각은 슈트를 오버클럭하고 개조하는 연산 자원으로 치환될 수 있습니다. 업그레이드를 많이 할수록 캐릭터의 피지컬은 강해지지만, 과거에 겪었던 루프의 기억과 진 엔딩 공략 힌트는 지워집니다.
*   **조각을 사용하지 않고 '보유'하면 기억이 선명해집니다**:
    성능 강화를 포기하고 '미사용 기억의 조각'을 인벤토리에 많이 누적할수록, 과거 루프에서 획득한 데이터가 동기화되어 인게임 데자뷔 대사가 자주 활성화되며, 비밀방 및 히든 아이템의 공략 단서가 온전하게 텍스트로 복원됩니다.

> [!CAUTION]
> **해커 NPC '바이츠(Bytes)'의 영구 업그레이드 시 경고**:
> *"기억의 조각을 슈트 연산 회로에 납땜하겠다고? 성능은 짱짱해지겠지만, 그 대가로 네 머릿속을 스치던 과거의 실마리는 연소되어 영영 사라질 거다. 강인한 깡통 로봇이 되어 무한히 오를 것인지, 나약한 인간으로 남아 진실의 문을 열 것인지는 네 선택이야."*

### ⚙️ 미사용 기억의 조각 보유량에 따른 기믹 연출 단계

| 보유 단계 | 미사용 조각 개수 | 슈트 전투 능력 | 인게임 데자뷔 & 스토리 힌트 수준 |
| :--- | :---: | :---: | :--- |
| **1단계: 완전 소실** | 0 ~ 2개 | **매우 강함** (업그레이드 올인) | 데자뷔 대사 없음. 비밀방 힌트 및 아이템 공략 지문이 심각한 글리치 노이즈(`[ERR_0x9A]`)로 파손되어 읽을 수 없음. |
| **2단계: 잔상 인식** | 3 ~ 6개 | **보통** | 단편적인 노이즈가 낀 대사가 출력됨. 힌트의 단어 일부분만 식별 가능. |
| **3단계: 기억 동기화** | 7개 이상 | **매우 약함** (노 업그레이드/하드코어) | 모든 데자뷔 대사가 명확하게 출력됨. 비밀방 기믹 조건 및 3종 히든 아이템('고장난 스틱', '수리키트', '사용방법')의 구체적인 획득 방식과 조합 힌트가 완전한 텍스트로 노출됨. |

## 4. ⚙️ 개발 구현 가이드 (Implementation Guide)

1.  **기억 조각 토큰 상태 분기 구현**:
    *   기억의 조각은 `localStorage` 상에서 `total_fragments`와 `spent_fragments`로 구분하여 관리합니다.
    *   `unused_fragments = total_fragments - spent_fragments` 공식을 사용하여, 미사용 기억 조각의 실시간 개수를 도출합니다.
2.  **프론트엔드 텍스트 노이즈(글리치) 필터**:
    *   인게임 스크립트 출력 모듈에서 `unused_fragments`가 3개 미만일 경우, 텍스트의 일정 비율(예: 40%)의 문자를 무작위 특수문자(`&`, `#`, `*`, `%`, `_`)로 변환하여 렌더링하는 글리치 함수를 적용합니다.
    ```javascript
    function applyGlitchFilter(text, unusedFragments) {
        if (unusedFragments >= 7) return text; // 3단계: 완전 복원
        let noiseRate = unusedFragments >= 3 ? 0.15 : 0.45; // 2단계 vs 1단계 노이즈율
        return text.split('').map(char => {
            if (char !== ' ' && Math.random() < noiseRate) {
                const noiseChars = ['@', '#', '$', '%', '&', '*', '_'];
                return noiseChars[Math.floor(Math.random() * noiseChars.length)];
            }
            return char;
        }).join('');
    }
    ```
3.  **데이터 바인딩 및 해킹 연출**:
